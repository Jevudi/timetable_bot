Для установки всех необходимых библиотек пропишите в консоли следующие комманды
pip3 install requests
pip3 install lxml
pip3 install pdf2image
pip3 install opencv-python
pip3 install numpy
pip3 install datetime

Также необходимо скачать poopler сделать это можно здесь:
https://poppler.freedesktop.org/
